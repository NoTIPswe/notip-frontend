Applicazione web di frontend sviluppata in Angular per il sistema NoTIP.

## Comandi principali

Assicurati di avviare il progetto all'interno del devcontainer per avere l'ambiente preconfigurato.

- **Avvio in locale:** `npm start` (esposto sulla porta 4200)
- **Build di produzione:** `npm run build`
- **Linting & Formattazione:** `npm run lint` e `npm run format`
- **Test & Coverage:** `npm run test:cov` (esegue Vitest e genera il report per SonarCloud)
